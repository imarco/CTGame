3G流量大转盘2期接口定义
=================
接口输入输出的数据采用json格式，不存在的参数，值输入空（''）


### 检查更新

*request*
    {
		type: 'CHECKUPDATE', //请求类型
		sessionId: '',
		versionId: '', //版本ID
		phone: '', //手机号
		data: {
		}
	}

*respones*
    {
		returnCode: '000', //返回代码
		message: '请求成功', //返回信息描述
		data: {
			versionCode: '', //手机号
			isUpdate: '', //是否强制更新
			updateUrl: '', //更新地址
			updateContent: '' // 更新内容
		}
	}
